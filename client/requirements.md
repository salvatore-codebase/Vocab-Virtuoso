## Packages
framer-motion | Complex animations for the hangman character and game states

## Notes
- Game logic handles 3 difficulty levels: Easy (8), Medium (6), Hard (4).
- Backgrounds are CSS-based dynamic scenes to keep assets lightweight.
- Hangman character is built using composed SVG/Div elements animated with Framer Motion.
